<h2>
Running Instructions
</h2>
<ol> 
<li> Download and extract the zip file
<li>Type <i> make</i> or <i> make assignment </i>.
<li> Run <i> ./assignment </i>.
<li>Run <i> make clean</i> to delete the unix executable file.
</ol>

<h2>
Assignment Explanation
</h2>
<p>

We have implemented a collection of basic integer arithmetic algorithms to find the square root of any number and the digits of pi upto any decimal places required.

We have implemented Toom's 3 for multiplication.
